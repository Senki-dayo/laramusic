## Laramusic
3人チームでの2週間創作。  
SpotifyのAPIを利用して作成した音楽主体のSNS。

- <b>ダッシュボード</b>  
直近に追加した曲を参照して、ユーザにおすすめの曲を一覧表示。
- <b>プレイリスト</b>  
追加した曲を一覧表示。タグの付け外し、タグの絞り込み。
- <b>タイムライン</b>  
自分とフォローしているユーザの最近のアクティビティを一覧表示。
- <b>タグ管理</b>  
タグや設定や削除など。
- <b>トラック検索</b>  
キーワードから曲を検索して結果を一覧表示。
- <b>ユーザ検索</b>  
キーワードからユーザを検索して一覧表示。
- <b>ユーザページ</b>  
ユーザ情報の表示。名前やメールアドレスの変更。

デザインを凝る時間はなかった。。
